# criacaoSenha
Criador e validador de senhas

A senha inserida pelo usuário para por uma série de testes para ver se é uma senha válida ou não para uso.
As validações são:
  -> Senha com 14 caracteres ou mais.
  -> Senha contêm letras maiúsculas e minúsculas.
  -> Senha contêm números.
  -> Senha contêm caracteres especiais.

Usando essas validações podemos identificar qual das características de senha faltam e já é retornado um aviso para o usuário,
fazendo com que ele possa alterar a senha.

Com todos os pontos OK a senha é criada.
